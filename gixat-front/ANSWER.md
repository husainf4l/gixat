## How It Works

1. **User logs in** → Assigned a role (admin, owner, or client)
2. **Sidebar appears** → Shows only features for that role
3. **Click menu item** → Navigates to feature page
4. **On feature page** → GraphQL queries will fetch data from backend
5. **Display data** → Show results on page
